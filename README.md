# pygames
Simple example cli games written in Python

# Get Started
1. Clone the repo
`git clone https://github.com/marktreble/pygames.git`
2. Enter the directory
`cd pygames/rock-paper-scissors`
3. Run the program
`python .` or `python3 .` (depending on how your environment is set up)

The games get progressively more complex, starting simple with rock-paper-scissors, hangman and noughts-and-crosses.
